# 📌 Descripción del proyecto

**EcoTravel Ecuador** es una propuesta de interfaz web interactiva desarrollada con **React.js** para una plataforma de turismo sostenible.

El proyecto presenta un Home moderno enfocado en promocionar los principales destinos naturales y culturales del Ecuador, brindando al usuario una experiencia visual atractiva, intuitiva y adaptable a diferentes dispositivos.

La aplicación permite explorar destinos turísticos, consultar información climática en tiempo real y conocer servicios relacionados con turismo ecológico.
